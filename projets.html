Mes projets en statistique

Analyse d'une relation macro-économique

J'ai réalisé un projet de macro-économie sur la relation entre PIB et chômage en Allemagne depuis 1991. J'ai pu appeler les bases de données de TAC economics via API sur R. J'ai ensuite fait le parallèle avec la loi d'Okun et réalisé plusieurs tests de régression afin de chercher s'il y avait une relation ou non, et si oui, si elle était positive ou négative.

Utilisation d'API via Python

En duo et pour un travail de faculté, nous avons réalisé un projet consistant à récupérer l'API de Doctolib pour trouver le centre de vaccination Covid le plus proche. Nous y avons ajouté l'API du STAR (Réseau de Transport Rennais) afin de signaler l'arrêt de bus le plus proche de l'utilisateur via la récupération d'IP. Ce code nous affichait une carte avec la position de l'utilisateur, le centre de vaccination le plus proche et l'arrêt de bus pour s'y rendre.
